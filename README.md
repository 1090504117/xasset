# XASSET 4.0

为你提供更精简、高效、安全的Unity项目资源管理方案。

- 主页：<https://www.xasset.org>
- Github：<https://github.com/xasset/xasset>
- Q群交流：[693203087](https://jq.qq.com/?_wv=1027&k=5DyV09a)

## 新特性

- VFS 让 IO 性能和安全性得到显著提升
- 大小包版本更新，指纹校验，异常处理更完善也更精炼
- 更智能的寻址模式与更可控的GC
